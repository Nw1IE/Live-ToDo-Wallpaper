# Live ToDo Wallpaper

**Live ToDo Wallpaper** — это интерактивные живые обои для Windows, которые выводят полноценный ToDo-список и панель с заметками прямо на твой рабочий стол поверх красивого видеоряда.

Проект работает на базе локального веб-сервера **Node.js**, который берет зацикленный медиафайл из папки, накладывает на него твою HTML-разметку интерфейса и транслирует готовый результат на экран. Твои задачи всегда перед глазами, а на фоне крутится выбранный ролик.

---

## Быстрый старт (Автоматическая установка)

В репозитории подготовлен специальный скрипт для автоматической сборки. Тебе не нужно вводить команды вручную:

1. Скачай или склонируй этот репозиторий к себе на ПК.
2. Найди в корневой папке файл `install.bat`.
3. Запусти его двойным кликом. Скрипт сам проверит окружение, установит необходимые зависимости и подготовит проект к работе.

---

## Ручная установка и запуск

Если ты хочешь запустить и настроить всё самостоятельно, следуй этим шагам:

### 1. Предварительные требования
Убедись, что в твоей системе установлен:
* **Node.js** (версии 18.0 и выше)

### 2. Установка зависимостей
Открой терминал (PowerShell или CMD) в корневой папке проекта и выполни:
```bash
npm install
```

## Тестовый запуск сервера
```bash
npm run dev
```
После этого открывай в браузере ссылку http://localhost:3000 — там отобразится твоя страница с заметками на фоне видео

## Настройка автозапуска
Чтобы сервер Node.js поднимался автоматически в скрытом фоновом режиме при включении компьютера (без висящих окон консоли), используется менеджер процессов pm2
1. Установи pm2 глобально в систему:
```bash
npm install pm2 -g
```
2. Настрой интеграцию с автозапуском Windows:
```bash
npm install pm2-windows-startup -g
pm2-startup install
```
3. Запусти фоновый процесс сервера:
```bash
pm2 start js/index.js --name "live-wallpapers"
```
4. Сохрани список процессов, чтобы Windows запускала бэкенд сама при старте системы:
```bash
pm2 save
```

## Как вставить свои фоновые видео?
Проект подтягивает видеопоток напрямую из локальных файлов. Чтобы заменить фон, тебе не нужно менять код:

1. Подготовь нужное видео. Рекомендуемый формат — .mp4, разрешение — под твой монитор (например, 1920x1080). Желательно использовать зацикленные (loop) видео.

2. Перейди в папку video/ в корне проекта.

3. Удали текущий файл video.mp4 (или переименуй его).

4. Закинь туда свой ролик и переименуй его строго в video.mp4.

5. Обнови страницу в браузере или перезапусти фоновый процесс, чтобы применить изменения.

## Структура проекта

├── js/
│   └── index.js       # Главный файл сервера (Node.js + Express)
├── video/
│   └── video.mp4      # Локальный фоновый видеоряд для живых обоев
├── index.html         # Семантическая разметка интерфейса заметок и ToDo-листа
├── style.css          # Стили оформления (стеклянный эффект Glassmorphism)
├── package.json       # Конфигурация проекта, скрипты и зависимости
└── .gitignore         # Исключения Git (чтобы не пушить тяжелые видео на GitHub)

## Как это устроено:
# Backend (js/index.js): Поднимает веб-сервер на базе express (порт 3000) и раздает статику. Он связывает медиапоток из папки video/ с интерфейсом фронтенда.

# Frontend (index.html / style.css): Отвечает за то, как именно будут выглядеть сами заметки. Разметка построена на чистых семантических тегах (<section> вместо <div>). Сверху накладываются стили с эффектом матового стекла (Glassmorphism), создавая современный UI, который не перекрывает фоновое видео.

# Хранилище: Все тудушки и записи сохраняются локально, поэтому они не пропадут при перезагрузке ПК или остановке сервера.

## Управление фоновым процессом (PM2)
Для контроля работы сервера в фоне используй следующие команды в консоли:

```bash
pm2 status — Проверить, запущен ли сервер прямо сейчас.

pm2 logs live-wallpapers — Посмотреть логи и ошибки сервера в реальном времени.

pm2 stop live-wallpapers — Временно остановить фоновый сервер.

pm2 delete live-wallpapers — Полностью удалить проект из автозагрузки.
```

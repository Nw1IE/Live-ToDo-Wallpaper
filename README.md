 Live ToDo Wallpaper
Live ToDo Wallpaper — это интерактивные живые обои для Windows, которые превращают твой рабочий стол в полноценный ToDo-список и панель с заметками. Проект работает на базе локального Node.js сервера, а отображение веб-страницы на рабочий стол обеспечивается утилитой Lively Wallpaper.

 Быстрый старт (Инструкция по запуску)
1. Предварительные требования
Убедись, что на твоем компьютере установлены:

Node.js (версии 18 и выше)

Lively Wallpaper (можно бесплатно скачать из Microsoft Store)

2. Установка зависимостей
Открой терминал (PowerShell или CMD) в папке проекта и установи необходимые пакеты:

Bash
npm install
3. Запуск сервера в режиме разработки
Для проверки работы скрипта и редактирования кода запусти локальный сервер:

Bash
npm run dev
После этого открой в браузере ссылку http://localhost:3000 — там должна появиться твоя страница с заметками.

 Настройка автозапуска (Чтобы всё работало само)
Чтобы сервер запускался автоматически в фоновом режиме при включении компьютера, используется менеджер процессов pm2.

Установи pm2 глобально в систему:

npm install pm2 -g

2. Установи модуль для интеграции с автозапуском Windows:
   ```bash
npm install pm2-windows-startup -g
pm2-startup install
Запусти проект в фоне:

pm2 start js/index.js --name "live-wallpapers"

4. Сохрани список процессов, чтобы Windows поднимала их сама:
   ```bash
pm2 save
 Установка на рабочий стол через Lively Wallpaper
Открой программу Lively Wallpaper.

В верхнем меню нажмите кнопку Добавить обои (+).

В поле Укажите URL-адрес вставь адрес запущенного локального сервера:

http://localhost:3000

4. Нажми кнопку со стрелкой (Далее), дождись загрузки превью, укажи название проекта и нажми **ОК**.
5. В настройках Lively Wallpaper убедись, что включен параметр **«Запускать вместе с Windows»**.

теперь при каждом старте системы Node.js сервер поднимется в фоне, а Lively Wallpaper натянет актуальный ToDo-лист на твой рабочий стол.

##  Структура проекта и Документация

```text
├── js/
│   └── index.js       # Главный файл сервера (Node.js + Express)
├── video/
│   └── video.mp4      # Локальный фоновый видеоряд для живых обоев
├── index.html         # Разметка интерфейса заметок и ToDo-листа
├── style.css          # Стили оформления (интерфейс, стеклянный эффект)
├── package.json       # Конфигурация проекта и скрипты запуска
└── .gitignore         # Список исключений для Git (Тяжелые видео, лог-файлы)
Архитектура и модули:
Backend (js/index.js): Написан на Node.js с использованием ESM ("type": "module"). С помощью фреймворка express поднимает веб-сервер на порту 3000 и статически раздает файлы фронтенда (index.html, style.css, скрипты и медиа).

Frontend (index.html / style.css): Веб-интерфейс, который рендерится поверх рабочего стола. Использует современную семантическую разметку и стили для адаптивного отображения поверх видеопотока.

База данных / Хранилище: Данные ToDo-листа и заметок сохраняются локально (через localStorage браузера или синхронизируются с бэкендом), что гарантирует сохранность записей при перезагрузке ПК.

 Полезные команды для управления
pm2 status — проверить, работает ли фоновый сервер прямо сейчас.

pm2 logs live-wallpapers — посмотреть логи сервера и ошибки, если что-то пошло не так.

pm2 stop live-wallpapers — временно остановить фоновый сервер.

pm2 delete live-wallpapers — полностью удалить проект из фонового автозапуска.

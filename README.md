# Live ToDo Wallpaper

**Live ToDo Wallpaper** — это интерактивные живые обои для Windows, которые выводят полноценный ToDo-список и панель с заметками прямо на твой рабочий стол поверх красивого видеоряда.

Проект работает на базе локального веб-сервера **Node.js**, который берет зацикленный медиафайл из папки, накладывает на него твою HTML-разметку интерфейса и транслирует готовый результат на экран. Твои задачи всегда перед глазами, а на фоне крутится выбранный ролик.

## Быстрый старт (Автоматическая установка)

В репозитории подготовлен специальный скрипт для автоматической сборки. Тебе не нужно вводить команды вручную:

1. Скачай или склонируй этот репозиторий к себе на ПК.
2. Найди в корневой папке файл `install.bat`.
3. Запусти его двойным кликом. Скрипт сам проверит окружение, установит необходимые зависимости и подготовит проект к работе.

## Настройка автозапуска
Чтобы сервер Node.js поднимался автоматически в скрытом фоновом режиме при включении компьютера (без висящих окон консоли), используется менеджер процессов pm2
1. Установи pm2 глобально в систему:
```bash
npm install pm2 -g
```
2. Настрой интеграцию с автозапуском Windows:
```bash
npm install pm2-windows-startup -g
pm2-startup install
```
3. Запусти фоновый процесс сервера:
```bash
pm2 start js/index.js --name "live-wallpapers"
```
4. Сохрани список процессов, чтобы Windows запускала бэкенд сама при старте системы:
```bash
pm2 save
```

## Как вставить свои фоновые видео?
Проект подтягивает видеопоток напрямую из локальных файлов. Чтобы заменить фон, тебе не нужно менять код:

1. Подготовь нужное видео. Рекомендуемый формат — .mp4, разрешение — под твой монитор (например, 1920x1080). Желательно использовать зацикленные (loop) видео.

2. Перейди в папку video/ в корне проекта.

3. Удали текущий файл video.mp4 (или переименуй его).

4. Закинь туда свой ролик и переименуй его строго в video.mp4.

5. Обнови страницу в браузере или перезапусти фоновый процесс, чтобы применить изменения.

## Структура проекта
```text
├── js/
│   └── index.js       # Главный файл сервера (Node.js + Express)
├── video/
│   └── video.mp4      # Локальный фоновый видеоряд для живых обоев
├── index.html         # Семантическая разметка интерфейса заметок и ToDo-листа
├── style.css          # Стили оформления (стеклянный эффект Glassmorphism)
├── package.json       # Конфигурация проекта, скрипты и зависимости
└── .gitignore         # Исключения Git (чтобы не пушить тяжелые видео на GitHub)
```

## Как это устроено

* **Backend (`js/index.js`)** Поднимает локальный веб-сервер на базе **Express** (порт `3000`) и раздает статические файлы. Он связывает медиапоток из папки `video/` с интерфейсом фронтенда.

* **Frontend (`index.html` / `style.css`)** Отвечает за визуальное отображение заметок. Разметка полностью построена на чистых семантических тегах (используются элементы `<section>` вместо стандартных `<div>`). 
  
* **Интерфейс (Glassmorphism)** Поверх фонового видео накладываются современные стили с эффектом матового стекла, создавая стильный UI, который отлично читается и не перекрывает анимацию.

* **Хранилище данных** Все задачи и записи сохраняются локально. Данные не пропадут при перезагрузке ПК, очистке кэша или остановке сервера.

## Управление фоновым процессом (PM2)

Для контроля и настройки работы сервера в фоновом режиме используй следующие команды в консоли:

```bash
# Проверить, запущен ли сервер прямо сейчас и сколько ресурсов он ест
pm2 status

# Посмотреть логи и ошибки сервера в реальном времени (если что-то не работает)
pm2 logs live-wallpapers

# Временно остановить фоновый сервер
pm2 stop live-wallpapers

# Полностью удалить проект из фонового автозапуска Windows
pm2 delete live-wallpapers
```

## Откат назад к стандартым обоям 
## Как полностью удалить проект и вернуть старые обои?

Если ты запустил проект просто «на тест» и хочешь вернуть свой прежний рабочий стол, а также полностью очистить систему от фоновых процессов, выполни эти простые шаги:

### 1. Выпиливаем сервер из автозапуска Windows
Чтобы скрипт Node.js больше не запускался сам при включении компьютера и не жрал ресурсы в фоне, открой терминал (CMD или PowerShell) и выполни одну команду:

```bash
pm2 delete live-wallpapers && pm2 save
Это полностью сотрет фоновый сервер из памяти pm2.

2. Возвращаем стандартные обои Windows
Так как проект подменяет обои на системном уровне через запуск локального медиапотока, для возврата твоей старой картинки сделай следующее:

Кликни правой кнопкой мыши по рабочему столу и выбери «Персонализация» (Personalization).

Перейди в раздел «Фон» (Background).

Выбери любые свои стандартные обои (фото или слайд-шоу). Windows сразу сбросит веб-окно сервера и вернет твой привычный бэкграунд.

После этого компьютер вернется в исходное состояние, как будто ты ничего и не устанавливал.
```
